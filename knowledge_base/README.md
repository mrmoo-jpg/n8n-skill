This directory is the knowledge base for the n8n Workflow Architect skill.

Drop your n8n documentation files into the appropriate subdirectories:

nodes/
  - One file per node type (e.g., webhook.md, agent.md, switch.md)
  - Include all parameters, operations, expression syntax, and known limitations
  - Use the node's official name as the filename

patterns/
  - Reusable workflow patterns (e.g., error-handling.md, agent-loop.md, rag-pipeline.md)
  - Include the node sequence, key configurations, and when to use the pattern

examples/
  - Complete example workflows with annotations
  - Can include exported JSON with commentary
